## Setup
Clone the project 
```
$ git clone  https://github.com/Vaibhav-84/Vcart.git 
```
To run the code on your computer

Open CMD in the same directory of manage.py
```
$ python manage.py runserver
```
Open your browser 
```
$ http://127.0.0.1:8000/ 
```
